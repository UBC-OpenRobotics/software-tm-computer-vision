## Usage
1. Setting up
   ```bash
   pip install git+https://github.com/facebookresearch/segment-anything.git
   pip install pycocotools onnxruntime onnx supervision
   ```
   
2. Download model
   ```bash
    wget https://dl.fbaipublicfiles.com/segment_anything/sam_vit_h_4b8939.pth
    ```
